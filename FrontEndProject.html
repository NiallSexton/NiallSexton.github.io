Nialls Board Game Reviews

In this project, I developed the front end aspect of my board games reviews project. Using React, HTML and CSS frameworks, a website was developed where a user could log into one of the profiles within the backend database and interact with the site. The user can then browse through the list of board games and once one has been selected, they can leave a vote or comment on the review. The website is still currently under development as new techniques and technologies are learnt and implemented.
